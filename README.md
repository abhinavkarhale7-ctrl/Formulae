# Formulae

Formulae is a darkroom chemical database designed for film labs and alternative process photographers.

It allows you to store, search, scale, and manage photographic chemical formulas directly in your browser.

No installation required.

## Features

• Store chemical formulas  
• Ingredient scaling  
• Process filters (B&W, C-41, ECN-2, E-6, etc.)  
• PDF and JPEG export  
• JSON backup and restore  
• Darkroom mode (red light UI)  
• Works offline  

## Use Cases

• Film labs managing chemistry recipes  
• Darkroom workers keeping formula notes  
• Wet plate photographers storing developers and collodion formulas
